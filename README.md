# 과제 수행 보고서: GitHub 홈페이지 및 실시간 시세 테이블 구축

## 1. GitHub 계정 및 저장소 생성
- GitHub 계정을 생성하고, `계정명.github.io`라는 이름의 Public 저장소를 생성하였습니다.

## 2. 데이터 수집 및 초기 작성
- CoinGecko 사이트에서 비트코인(Bitcoin)과 이더리움(Ethereum)의 실시간 시세를 확인하였습니다.
- 기초 데이터를 마크다운 표 형식으로 정리하였습니다.

## 3. HTML 변환 및 기능 구현
- LLM을 활용하여 마크다운 표를 HTML 구조로 변환하였습니다.
- 변환된 HTML에 CSS 스타일을 적용하여 시인성을 높였습니다.

## 4. 실시간 API 연동 (JavaScript)
- CoinGecko의 공개 API(`api.v3/simple/price`)를 활용하였습니다.

- `fetch` 함수를 사용하여 비동기적으로 데이터를 가져오고, `setInterval`을 이용해 30초마다 자동으로 테이블 가격이 갱신되도록 구현하였습니다.

## 5. 오류발생 및 수
- 현재 가격 (USD) 테이블에 오류 발생
- CoinGecko의 무료 API 호출 횟수 제한에 의한 일시적 차단
- API 호출이 실패하더라도 기본 가격이 표시되도록 코드를 보완
- 1~5분 이내로 오류 자동 해결
